## Local Install
### Prerequisites
- Node.js (v18 or higher)
- MongoDB
- npm or yarn

### Installation

Clone the repository
```bash
git clone <repository-url>
cd StreamSphere
```

**Backend:**
```bash
cd server
npm install
```

**Frontend:**
```bash
cd frontend
npm install
```
